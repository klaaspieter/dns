# DNS Configuration

My DNS configuration.

## Installation

Install [DNSControl](https://docs.dnscontrol.org/getting-started/getting-started#id-1.-install-the-software).

## Usage

Validate configuration:

```sh
dnscontrol check
```

Preview changes:

```sh
dnscontrol preview
```

Deploy changes:

> [!NOTICE]
> After careful review!

```sh
dnscontrol push
```
